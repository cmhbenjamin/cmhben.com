# Colophon
I build this website mostly for recording the more technical aspect of projects that is hard to show in a reel

Also for storing my notes and process when I work on project
I want this site to look simple and unique, avoid cookie-cutter UI components
Leave out planty of breathing room, only show necessary info

## Features
- Responsive design
- Edit content using Markdown
- Manage content with Frontmatter CMS in VSCode
- Emojis
